# Вычислитель отличий

#### Gendiff - утилита для вычисления отличий и их визуализации

#### Использование:

`gendiff [options] <filepath1> <filepath2>`

#### Опции:

```Options:
  -V, --version        output the version number
  -f, --format [type]  output format
  -h, --help           display help for command
```
